# eslint-config-mnmo

[Shareable eslint config](http://eslint.org/docs/developer-guide/shareable-configs) to be used by mnmo.com.br repos.

# Usage

```bs
npm install --save-dev eslint-config-mnmo
```

.eslintrc
```json
{
    "extends": "mnmo"
}
```
